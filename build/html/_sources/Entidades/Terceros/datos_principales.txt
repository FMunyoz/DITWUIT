*****************
Datos Principales
*****************
.. image:: ../../images/Entidades/tercero/datos_principales.png
   :scale: 75%
   :align: center
   
Desde la pantalla de datos principales es posible acceder y/o modificar la información básica de todo tercero.

- **Organización**. Es el código de la organización que va a interactuar con el tercero que estamos consultando y o creando. 
- **Tags**. Es posible asociar los terceros a diferentes clasificaciones que faciliten el posterior uso del tercero.
- **Codigo**. Es el código del tercero.
- **CIF**.
- **Nombre**.
- **GLN**. Es el código asociado por GS1 a este tercero.
- **Estado**. Indica si se siguen realizando intercambios con este tercero.
- **Direccion**. Permite indicar la información postal de este tercero.

Es posible asignar una imagen al tercero para identificarlo.
