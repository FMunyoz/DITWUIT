************************
Etiquetado. Introducción
************************

El etiquetado es un aspecto fundamental en la gestión de **DITWUIT**.

Toda la mercancía debe estar etiquetada a nivel de unidad logística para ser entrada, ubicada y preparada en el almacén.

Además, toda ubicación debe estar identificada por un código de barras para que se puede identificar dónde ubicar mercancía, de donde se coge,...

Dentro de esta opción podemos emitir:

.. index:: pair: Etiquetado; Etiquetas de contenedor vacío

- **Etiquetas de contenedor vacío**. Etiquetas sin contenido con un SSCC que son utilizadas en el proceso de :ref:`picking`.

.. index:: pair: Etiquetado; Duplicado de etiqueta de contenedor

- **Duplicado de etiqueta de contenedor**. Permite repetir la etiqueta de la unidad logística.

.. index:: pair: Etiquetado; Imprimir etiquetas de ubicación

- **Etiquetas de ubicación**. Etiquetas que identifican una ubicación del almacén.
