Datos Principales¶

Desde aquí es posible dar de alta y consultar la información de almacenes:

  • Código. Código asignado al almacén.
  • Nombre. Nombre del almacén.
  • Tercero. Tercero asociado al almacén.
  • Ruta de Picking. Ruta de picking asociada para el proceso de preparación.

También se pueden identificar las dimensiones del almacén.

Desde el almacén se tiene acceso a otras acciones:

  • Asistente Creación de almacén
  • Regenerar stock. A ejecutar en caso de encontrar alguna discrepancia entre el resumen de stock y el detalle.
  • Reparar contadores.
  • Foto de stock. Permite generar un archivo con la foto de stock.

Desde esta opción también es posible acceder a Estanterías, almacen-terceros, almacen-configuracion y almacen-eventos.
